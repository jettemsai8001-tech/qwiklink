Firebase Setup Guide 🔥
Follow these steps once to enable real authentication
Step 1 — Create Firebase Project
1. Go to console.firebase.google.com
2. Click Add Project → name it "QwikLink"
3. Disable Google Analytics (optional) → Create project
Step 2 — Enable Authentication
1. In Firebase console → Authentication → Get Started
2. Enable Email/Password provider
3. Enable Google provider → set support email
Step 3 — Get Your Config
1. Project Settings (⚙️) → General
2. Scroll to Your apps → Add web app
3. Copy the firebaseConfig object
// Paste into this file — replace YOUR_... values:
const FIREBASE_CONFIG = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT.appspot.com",
  messagingSenderId: "YOUR_SENDER_ID",
  appId: "YOUR_APP_ID"
};
Step 4 — Firestore Database (optional)
To save business page data:
1. Firestore Database → Create database → Start in test mode
2. Business data auto-saves when user completes wizard
